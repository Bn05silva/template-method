# Padrão Template Method

## 📌 Descrição
Implementação do padrão de projeto **Template Method** em Java, utilizando o tema de **Bebidas Quentes**.  
A classe abstrata `BebidaQuente` define o algoritmo genérico para preparar uma bebida, enquanto as subclasses `Cafe` e `Cha` personalizam os passos específicos da receita.
